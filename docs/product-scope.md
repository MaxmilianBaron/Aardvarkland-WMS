# Product Scope

Aardvarkland WMS focuses on practical warehouse operations.

## Warehouse Operations

- inbound receiving and putaway
- stock visibility by SKU and location
- picking tasks and outbound release
- packing, parcel handling, and shipment labels
- returns, quality checks, and quarantine
- cycle count plans and variance approval
- SLA, dock, and vehicle visibility for operational control

## Hardware Direction

- scanner-first workflows
- keyboard wedge support for USB and Bluetooth scanners
- Zebra DataWedge path for supported Android terminals
- ZPL label generation
- local Print Agent as the primary production print path
- scanner fleet health with battery, signal, last activity, and worker
  assignment

## Roles

- Warehouse worker: operational warehouse work, scanner flows, receiving,
  picking, packing, and allowed label printing.
- Warehouse manager: operational management, status, print queue, reprint, and
  exception handling.
- System administrator: users, roles, warehouses, integrations, printers, print
  agents, templates, and stability/reliability views.

## Operations

- installable RF/PWA shell
- frontend runtime event reporting
- OpenAPI/frontend contract checks
- readiness, alert, recovery, and incident visibility

## Localization

The product direction is Czech-first with English and Ukrainian as supported UI
languages.
